## ◈ Homelab

Designed and built a personal homelab using a repurposed laptop as a Proxmox VE server, managed from a dedicated admin machine over a home network - used to practice virtualization, remote administration, storage, and networking.

```
  ┌─────────────────────────────────────────────────────────────────────────────────┐
  │  SERVER      Lenovo IdeaPad 300(old laptop) · i5-6200U · 12 GB RAM · 500 GB SSD
  │  ADMIN       Lenovo Legion 5 · i7-14650HX · 32 GB DDR5 · 1 TB SSD
  │  NETWORK     NETGEAR Nighthawk R7000 · AC1900 · Gigabit LAN
  ├─────────────────────────────────────────────────────────────────────────────────┤
  │  HYPERVISOR  Proxmox VE — VM deploy · ISO install · snapshots
```
